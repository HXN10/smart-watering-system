üìù How to Set Up MongoDB (Persistent Storage)

  1. Start MongoDB using Docker Compose: 
    cd smart-watering-system
docker-compose up mongo
  2. Set the MONGO_URI environment variable: 
    export MONGO_URI="mongodb://localhost:27017/watering"
  3. Update the controllers to use Mongoose models instead of in-memory objects
  4. View data using MongoDB Compass or: mongosh mongodb://localhost:27017/watering
